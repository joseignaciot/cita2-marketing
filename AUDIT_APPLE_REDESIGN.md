## 🎨 Design System Apple-Style

### Colores
```
Primary: #0071E3 (Apple blue)
Secondary: #86868B (Apple gray)
Background: #FAFAFA (Soft white)
Text: #1D1D1F (Near black)
Accent: #34C759 (Success green)
Warning: #FF9500 (Orange)
```

### Tipografía
```
Display: 48-64px, font-weight 700
Headline: 32-40px, font-weight 600
Title: 24-28px, font-weight 600
Body: 17-19px, font-weight 400
Caption: 14-16px, font-weight 400
```

### Espaciado (Generoso)
```
Section: 120px vertical
Component: 60px gap
Element: 24px gap
Tight: 12px gap
```

### Animaciones
```
Entrance: 0.6s cubic-bezier(0.22, 1, 0.36, 1)
Hover: 0.3s ease
Stagger: 0.1s delay entre items
```
