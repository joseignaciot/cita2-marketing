# SEO y AI SEO - Estrategia Cita2 Marketing

## 📊 Resumen de Optimizaciones Implementadas

### 1. Schema.org Avanzado (Structured Data)

#### Schemas Implementados:
- **SoftwareApplication** - Producto principal con ratings, reviews, offers
- **Organization** - Datos completos de empresa, contacto, redes sociales
- **WebSite** - SearchAction para búsqueda interna
- **HowTo** - Guías paso a paso con supplies, tools, steps
- **FAQPage** - Preguntas frecuentes optimizadas para featured snippets
- **Service** - Servicios específicos por sector
- **LocalBusiness** - Para landings de sectores
- **Article** - Blog posts con author, datePublished, wordCount
- **BreadcrumbList** - Navegación estructurada
- **AggregateRating** - Valoraciones con bestRating/worstRating

#### Características AI-SEO:
- `@graph` structure para múltiples entidades relacionadas
- `@id` únicos para cada entidad (entity references)
- `potentialAction` para SearchAction
- `sameAs` para social profiles
- `contactPoint` con horarios de atención

### 4. Content Clusters (Pillar Pages)

#### Estructura de contenido:

**Pilar Principal:** `/` - Software de gestión de citas
- **Cluster Peluquería:** `/peluquerias` → `/blog/reducir-ausencias-peluqueria`
- **Cluster Estética:** `/estetica` → Guías específicas
- **Cluster Restaurantes:** `/restaurantes` → `/blog/aumentar-reservas-restaurante`
- **Cluster Turismo:** `/turismo` → Contenido sectorial
- **Cluster Comparativas:** `/vs/agendapro`, `/vs/reservio`

#### Internal Linking:
- Breadcrumbs en todas las páginas
- Links contextuales entre landings y blog
- "Artículos relacionados" al final de guías
- Cross-linking entre comparativas
